### 형식 2종 (열 구성이 다름 — 통합 필요)
| 변형 | 예시 파일 | 열 구성 | 정답 형태 |
|---|---|---|---|
| **내신어법** | G구성고1내신어법2단계, G동백고1… | A=번호 · D=문장 · E=정답 · F=단원(3-1) | **`O`**(맞음) 또는 **`wrong->right`**(고칠 것, 예 `had->has`) |
| **모의고사어법** | G고등어법12/34/45 | A=번호 · B=문장(`[..]`/`①~⑤`) · C=구분 · D=정답 | 교정형(`To avoid, Avoiding`) 또는 번호(`1`) |

- 내신어법 = "이 문장 어법상 맞나? 틀리면 고쳐라" (O / 교정).
- 학교 내신이 핵심이므로 **내신어법(O / wrong->right)** 우선.

## 2. 응시/채점 방식 — ✅ 확정 (2026-06-15)

**자동채점 최대한 → 교사 검수.** 학생은 문장별 "맞음(O)" 또는 "고친 형태"를 입력하고,
아래 엑셀 채점공식과 **동일한 로직으로 자동채점** 후 교사가 O/X 보정(요약문 채점 UI 재사용).

### 원본 엑셀 채점공식 (꼭 반영)
```excel
=IF(TRIM(F3)="","",
  IF(OR(
    ISNUMBER(SEARCH(","&N(C3)&",", ","&N(F3)&",")),
    ISNUMBER(SEARCH("-"&N(C3)&",", ","&N(F3)&","))
  ),"O","X"))
```
여기서 `N(x)` = `SUBSTITUTE`로 **공백 제거 + `=>`·`->`·`→` 를 모두 `-` 로 치환**. `C3`=학생 입력, `F3`=정답키.

- 의미: 정답키(F)를 `,…,` 로 감싸고, 학생답(C)이 **① `,학생답,` 완전 토큰 일치**(콤마 구분 다중정답 중 하나) 또는 **② `-학생답,` = 화살표 교정의 오른쪽(고친 형태) 일치** 면 **O**.
- 예) 정답 `had->has` → 정규화 `had-has`. 학생 `has` → `-has,` 가 `,had-has,` 안에 있음 → **O**.
- 예) 정답 `To avoid, Avoiding`(다중) → 학생 `Avoiding` → `,Avoiding,` 토큰일치 → **O**.
- 예) 정답 `O`(맞는 문장) → 학생 `O`/맞음 → 토큰일치 → **O**.
- `SEARCH`는 대소문자 무시 → 정규화 시 소문자화.
- 정답키(F)가 비면 채점 안 함(공란 행/채점불가 → 교사).

## 4. 자동채점 구현 (엑셀 공식 → Python, 그대로 반영)

```python
def _norm(s):
    """공백 제거 + =>·->·→ 를 - 로 통일. SEARCH가 대소문자 무시이므로 소문자화."""
    s = (s or '').replace(' ', '').lower()
    for a in ('=>', '->', '→'):
        s = s.replace(a, '-')
    return s

def auto_grade(student_input, answer_key):
    """엑셀 채점공식과 동일. 정답키 비면 None(채점 안 함). 그 외 True(O)/False(X)."""
    if not (answer_key or '').strip():       # =IF(TRIM(F3)="","")
        return None
    si = _norm(student_input)                # SUBSTITUTE(C3 ...)
    hay = ',' + _norm(answer_key) + ','      # ","&SUBSTITUTE(F3 ...)&","
    # (1) ,학생답, 토큰 완전일치   (2) -학생답, 화살표 교정 오른쪽 일치
    return (',' + si + ',') in hay or ('-' + si + ',') in hay
```
`SEARCH(x, hay)` = `x in hay`(부분문자열 포함), `OR` = `or`, `SUBSTITUTE 4중` = `_norm`. 공식과 1:1 대응.

## 4-1. 검수 워크플로우 — 자동채점 후 'X 위주' 교사 검수
1. **제출 즉시 전 문항 자동채점**(위 함수) → O/X 저장.
2. 교사 채점 화면은 **X(틀림) 위주로** 보여줌(요약문 grade_detail에 'X만 보기' 필터·X 먼저 정렬). O는 접고 X만 빠르게 훑음.
   - 목적: 오탐(학생이 맞는 변형을 썼는데 공식이 못 잡아 X) 만 골라 **O로 정정**.
3. 최종 점수 = 자동 O + 교사가 O로 바꾼 것. (교사 미검수여도 자동 O/X로 점수 즉시 산출)
