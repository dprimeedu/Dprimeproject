## 3. 영작 단원 업로드 강화

### 다중 파일 일괄 업로드
- 한 번에 여러 엑셀 선택 → 파일 1개 = 단원 1개
- 학년 / 출판사 / 단원명을 파일명에서 자동 추출 (`writing/services/excel.py::parse_filename`)
- 첫 매치 위치부터 파일명 끝까지를 단원명으로 (예: `1과 본문1`, `3과 Wrap Up`)

### 양 끝 노이즈 자동 정제 (`clean_prefix`)
- 회화 표지 `Q:` `A:` `B:` `G:` `Q.` `A.`
- 번호 `1.` `2.` `3)` 등
- 불릿/따옴표 `-` `•` `*` `"` `'`
- lookahead로 **단일 알파벳 토큰만** 매치 (`well.` 같은 단어 꼬리 보호)
- 정상 종결 `.?!`과 어퍼스트로피(`don't`) 보호

### 3단어 이하 필터
- 정제 후 영어 단어 수 ≤ 3 행은 자동 제외 (`skipped_short` 카운트)

### 중복 단원 skip
- 학년 + 출판사 + 단원명이 모두 같으면 skip + 안내

### AI 한글뜻 자동 시작
- 업로드 직후 새 단원들에 백그라운드 스레드 시작
- 표 "AI 한글뜻" 컬럼이 실시간 갱신

### 색인 컬럼 처리
- 엑셀의 색인 컬럼은 무시하고 항상 행 순서대로 `index` 자동 부여 → `UNIQUE constraint failed` 회피

### 엑셀 양식 다운로드
- `/admin/students/template.xlsx` (학생용: 색인·ID·이름)
- `/admin/template.xlsx` (영작 단원용: 색인·영어·한글)
- 헤더 굵게 + 컬럼 너비 설정 + 예시 행 3개

## 4. 학생 등록 시스템 — 학원이 직접 만든다

### Member 모델 변경 (migration `0002_member_is_approved_member_login_id_and_more`)
- `login_id` (unique, max_length=50) — 학원이 등록한 ID
- `is_approved` (BooleanField) — 재원생 승인 여부
- `email` → nullable (학원 등록 학생은 이메일 없음)

### Custom auth backend (`member/backends.py`)
- `LoginIdOrEmailBackend` — `login_id` 또는 `email` 둘 다로 로그인 가능

- 학원 가입: 기존대로 `is_active=False` (관리자 승인 후 로그인)

### 로그인 폼

## 9. 레벨 시스템 점진식 변경

이전: `total_xp // 100 + 1` (고정 100/레벨)
이후: 점진적 증가 (Lv N → N+1 필요 XP = `190 × N`)

| Lv | 이번 레벨 필요 | 누적 |
|---|---|---|
| 1 | 190 | 190 |
| 2 | 380 | 570 |
| 3 | 570 | 1,140 |
| 5 | 950 | 2,850 |
| 10 | 1,900 | 10,450 |
| 20 | 3,800 | 39,900 |

- `XP_PER_LEVEL_UNIT = 190`
- `level_start_xp(N) = 190 × N × (N-1) / 2`
- `compute_level(xp)` — 수학식 (sqrt) + 부동소수 보정
- `StudentProfile.level/xp_in_current_level/xp_to_next_level` 모두 scoring 함수 사용으로 통일
- 호칭(title) 임계값은 그대로 Lv 기준 — 호칭 도달이 자연히 느려짐

## 11. 발견·정리한 데이터 이슈

### 학생 일괄 삭제 500 — 누락 테이블
- 원인: cascade collector가 `enrollments` / `translation_log` 테이블 조회 시도 → SQLite "no such table"

- `Uncaught SyntaxError: Identifier 'searchEl' has already been declared`
- 학생 모달 검색 변수와 단원 검색 변수가 동일 함수 스코프에서 중복
- 학생 모달 → `modalSearchEl`로 rename

### Bootstrap is not defined
- 인라인 script가 본문 중간에 즉시 실행됐는데 Bootstrap JS는 body 끝에서 로드
- `IIFE` → `DOMContentLoaded` 이벤트로 감싸기

### student_list_api 500
- `u.first_name`, `u.last_name` 참조 — 우리 Member는 AbstractBaseUser 기반이라 그 필드 없음
- `getattr(u, 'name', None) or u.username` 폴백으로 변경

### UNIQUE constraint failed: writing_problem
- 엑셀 색인 컬럼 중복 또는 공백
- 색인 컬럼 무시 + 행 순서대로 자동 인덱스 부여로 해결

### NAS gunicorn 502
- 잦은 commit + auto-deploy 재시작 사이에 요청 들어가서 일시적 다운타임
- 백그라운드 AI 한글뜻 스레드가 재시작 시 끊김 → 일괄 보충 버튼으로 대응
