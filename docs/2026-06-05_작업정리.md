## 1. 낱말카드(단어찾기) 별표 기능 추가 — 배포 완료
기존에는 별표가 **단어훈련 플래시카드(`StudentWordStar` / VocabWord)**에만 있었음. 어제 작업으로 **단어찾기로 만든 낱말카드(`WordCard`)에도 학생별 별표**를 달 수 있게 확장.

### 모델
- `WordCardStar` 신규 (`vocab/models.py:249`): 학생 ↔ 낱말카드 별표.
  - FK `student`(related_name `wordcard_stars`), `card`(related_name `stars`), `created_at`.
  - `unique_together = (student, card)`, 인덱스 `(student, card)`, `db_table='vocab_wordcard_star'`.
- migration `0006_wordcardstar` (deps: 0005_vocabunit_category).

### API
- `wordcard_star_toggle_api` 신규 (`vocab/views.py:167`): body `{word_id, starred}` → `WordCardStar` get_or_create / delete.
  - 소유권 검사: `WordCard` pk + `card_set__student=request.user`로 본인 카드만 토글.
- URL `api/wordcard-star/toggle/` → name `wordcard_star_toggle` (`vocab/urls.py:29`).
- 기존 단어 별표 API(`star_toggle_api`, `api/star/toggle/`)는 그대로 유지 — 두 종류 별표 병존.

### 플래시카드 학습 연동
- 컨텍스트에 `wordcard_star_url` 추가하여 낱말카드 학습 시 별표 토글 가능 (`vocab/views.py:1085`, `1230`).
- 별표 개수/목록 집계 추가: `wc_star_count` (`views.py:1039`), `wc_stars` 목록 (`views.py:1055`), 학습 진입 시 별표 카드 조회 (`views.py:1213`).
- 템플릿 `flashcard.html`: 단어 별표 URL(`STAR_URL`)과 낱말카드 별표 URL(`WC_STAR_URL`, line 218) 분기 처리. 별표만/별표 제외 토글, ☆ 버튼 동작은 기존 로직 재사용.

### 화면
- `vocab/templates/vocab/wordcard_list.html`, `vocab/templates/vocab/home.html`, `config/templates/training.html` 수정 (별표 진입/표시 반영).
- `config/settings.py` 소폭 수정.
