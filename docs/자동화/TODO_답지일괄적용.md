## 다음 작업 — 변형문제 자동 균형 생성

### 배경
순환횟수 1/1 설정 시 비변형 167 vs 변형 79 — **불균형**.
변형문제가 없는 (단원, 유형) 조합이 존재하기 때문.

### 목표
③ 변형문제 생성 단계에서:
1. **갭 분석**: 비변형은 있지만 변형이 없는 (단원, 유형) 조합 자동 탐지
2. **AI 자동 생성**: 해당 지문을 기반으로 변형문제 자동 생성 (Gemini/Claude)
3. **변형문제 시트에 삽입**: 올바른 형식 + 유형 태그로
4. 단순 개수 채우기가 아닌 **번호별, 유형별 매칭**

### 갭 분석 로직 (구현 필요)
```python
# 비변형이 있는 (유형, 단원) 조합
nv_combos = {(type, unit) for q in 비변형_questions}
# 변형이 있는 (유형, 단원) 조합  
v_combos = {(type, unit) for q in 변형_questions}
# 갭 = 비변형만 있고 변형 없는 조합
gaps = nv_combos - v_combos
# 각 gap에 대해: 해당 지문 읽기 → AI로 변형문제 생성
```
