## 2. 빨파정답 옵션 동작 (`CLASS/고등교과서자동화.py`)
- `excel_rb`(내신빨파)에서 **빨파정답 복사부를 `excel_rb_answer`로 분리**.
- **빨파정답 옵션 = ① `excel_rb_answer`**(정답 이미지/PDF 복사) **+ ② `excel_rb_analysis`**(이미 만들어진 '상세분석+빨파정답(내신용)' PDF를 **선택 단원만** 출력 폴더로 복사). **AI 재실행 아님.**
- `excel_rb_analysis` 소스: `xlsx폴더\{교재} 상세분석+빨파정답 (내신용)\{교재} 상세분석+빨파정답 {단원}번(내신용).pdf` → 출력: `ANSWERPATH\{이름} 상세분석+빨파정답\`
- ⚠️ 그 (내신용) PDF는 **영어분석지로 미리 생성**돼 있어야 함. 없으면 "먼저 영어분석지를 생성하세요" 안내.

## 3. 영어분석지(EngAnalysis) — 빨파 소스 확장 (`CLASS/EngAnalysis/`)
- `run.py` `_find_known_phrases` 우선순위:
  1. ①-a F열(정답) PDF(형광펜 주석) → fitz
  2. **①-a2 F열이 직접 이미지(png/jpg)면 그 이미지를 비전으로** (신규) — `원문+빨파(정답)\...{단원}.png` 처럼 컨벤션과 다른 폴더/이름도 동작
  3. ①-b 컨벤션 `{교재} 빨파정답\{교재} {단원} 빨파정답.pdf`
  4. ② 통합 PDF `{교재} 원문+빨파(정답|문제).pdf`
  5. ③ 컨벤션 번호별 PNG(비전)
  6. 없으면 AI 추정
- `is_image_file` 은 `pdfify`에 있음(주의: `pdf_reader` 아님).
- **`--force-api`가 `--naesinyong` 출력 경로(내신용 폴더)를 무시하던 버그 수정** (`_auto_save_rel(..., naesinyong=args.naesinyong)`).
- 제어문자(`\x01`, HWP→PDF artifact) 정규화.

## 4. 답지 네트워크 이동 (`_교과서자동화_원본/a. 고등교과서자동화.py` `moveAnswerToDir`)
- 기존 **no-op** → **부교재와 동일하게** 정답 폴더(`self.answerPath`) 전체를
  `어서와\답지\내신\{년도}년 {시험}\{학교}\{이름} 정답` (= **`Y:\답지\내신\...`**) 로 복사. 로컬 정답 폴더는 유지.
- 학교/시험명은 `SCHOOLREGEX`/`TESTREGEX`로 입력 이름에서 추출. `PATH.getPath("ANSWERPATH")` → `Y:\답지\내신`.

## 5. 파일병합 (`CLASS/고등교과서자동화.py` `merge_pdf`)
- **상세분석+빨파정답 개별 PDF를 단원순으로 한 파일** `{이름} 상세분석+빨파정답.pdf` 로 병합해 **SAVEPATH(part 폴더)** 에 저장. 학생 통합본(`{이름} 통합.pdf`)에는 제외.
- **책표지(해당 part) 못 찾을 때 `part_regex.search(...).group()` None 크래시** → 통합본만 저장 + 안내로 수정.

## 6. copy_excel (`CLASS/고등교과서자동화.py`)
- `'단원'` 열 없는 시트(예: `출력이력`)는 건너뛰도록 수정 (`KeyError: '단원'` 방지).

## 남은 일 / 주의
- **DL 단원처럼 빨파정답 PNG가 없는 단원**은 AI 추정으로 생성됨(빨파 부정확) → PNG 만들고 재생성 필요.
- **part별 책표지 없으면** 책 형태 통합 안 됨(통합본만 나옴) → 해당 part 책표지를 책표지 폴더에 넣기.
- 실제 GUI 동작(Tkinter+HWP)은 직접 띄워서 확인 권장.
