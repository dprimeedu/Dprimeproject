## 지금 가능한 것
- **웹 일일입력 그리드(`/report/`)** 에서 출석·4과목결과·과제달성률·숙제순위·독해/문법과제·오늘/다음 과제·단어장·한마디 등 **대부분 바로 입력 가능**.
- 입력하면 **현황판(`/report/study/`)·학생 리포트**에 반영됨.
- 단어 오늘 볼 TEST(범위/합격)는 학생이 풀면 **자동**으로 현황판에 뜸(수기 불필요).

## 할 일 (⚠️ 미구현 — 나중에 하나씩)
나중에 추가할 때 각각 필요한 작업:

- [ ] **요일시간**: 학생별 시간표 — StudentInfo에 필드 추가(권장) 또는 그리드 노출
- [ ] **TEST 링크 / 독해 링크1·2**: 학생/과제별 외부 링크 필드(DailyRecord에 `test_link`, `reading_link1/2` 추가 + 그리드)
- [ ] **오늘문법과제**: `today_grammar` 필드 추가(현재 `next_grammar`만 있음)
- [ ] **각 과목 "단위"·시험명**: 어법단위/영작단원/TEST시험명 등 — 필요 시 DailyRecord에 필드 추가
- [ ] **어법(문법) 시험**: 웹에 어법 TEST 앱 자체가 없음 → 단어/요약문/영작/시험처럼 별도 앱 또는 결과 텍스트칸만이라도
- [ ] **(선택) 엑셀 → 웹 자동 동기화**: `0. 학생관리자료.xlsx[학생관리표]` 한 행을 읽어 DailyRecord로 밀어넣는 스크립트(단어TEST 범위 동기화 패턴 재사용). 이거 만들면 엑셀 그대로 쓰면서 웹에도 전부 반영.
