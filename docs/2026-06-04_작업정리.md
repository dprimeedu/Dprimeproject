## 1. 단어훈련(vocab) UX 개편 — 배포 완료
- **번호 세트 = 플래시카드 시험**으로 단일화: 단원 카드의 보라색 "테스트(영→한)" 제거, 100단어 번호 버튼 클릭 → 바로 플래시카드 시험. 번호 버튼 3열 격자 정렬.
- 한글 입력 시험(range_test)은 **진입점만 숨김**(코드 보존).
- 플래시카드: **별표만 / 별표 제외** 토글, 카드 높이 62vh로 확대.
- **답 노출 방지**: 한글(뒷면) 상태에서 다음 카드 넘길 때 애니메이션 없이 즉시 앞면(영어).
- **학습 현황 관리에서 "모름"으로 넘기면 자동 별표**(이미 별표면 유지, undo 시 자동별표만 해제).
- 번호 세트 진입 기본값: **카드섞기 + 별표만**(별표 있을 때).

## 2. 단어찾기(낱말카드 만들기) 신규 — 배포 완료
- 퀴즈렛식 낱말카드 세트 생성: 단어(영어) 입력 → 뜻(한글) 자동 채움.
- **사전 조회 순서**: 전체 단어장 사전(DictionaryEntry) → 캐시 → 교재 단원 → **Gemini AI 폴백**.
- 번호 연속 매김, 카드 수/제목 자동, 자동 임시저장, 완성 시 플래시카드 학습.
- **통합 사전 적재**: '단어장 전체 영영전체모음.xlsm'(B영어/C한글) 18,096개 + `TEST 자동화\단어` 폴더 단어장 53개 합쳐 **운영 사전 약 2.1만 개**. 단어 import 시 자동 합치기(`import_dictionary --from-vocab`, `import_vocab` 자동 동기화).
- 모델: `DictionaryEntry`, `WordCardSet`, `WordCard`, `DictionaryCache` (migration 0003·0004).

## 4. 중요영작 "O" 표시 (서술형 추출) — 데이터 작업
- 직전보강 HWP(`pyhwp`로 표 파싱)에서 "서술형 대비 핵심 문장" 추출.
- `2026년 영어영역 통합.xlsx` 중요영작 F열 **O 213개**, `2026년 영어독해 통합.xlsx` **O 204개**(한 글자 오타·인코딩 깨짐은 동일 처리, 안내문/도표 제외).
- 영어독해 **D열 양식 통일**: `1 No. 1` → `1-1`(1,708행).
- ⚠ 중요영작 시트 **영어 열이 파일마다 B/C 다름** → 한글 포함 여부로 자동 감지. 저장은 **해당 시트 XML만 패치**(다른 수식 시트 캐시 보존), 백업 생성.

## 5. 영작훈련 — 고등 문장 자동 분할 (업로드 시) — 배포 완료
