## 1. 오류수정양식 → 학생 시트 오류 표시  (신규)
- **오류표시.py** (`TEST 및 답지\일괄출력\TEST 자동화\오류표시.py`): 오류수정양식 시트의 오류를 학생 구글시트 Part1~4에 표시.
  - **K열('시험') 필터 필수** — 옛 데이터(K 비어있음) 자동 제외. 현재 시험명은 월 기준 자동(6월→1학기 기말, `B. 부교재 출력.py` checkCurrentSeason 로직).
  - 교재명 `partN`→PartN, 교재번호=Part 시트 번호.
  - **정답교정**: 학생에게 안 보이게 **숨김 I열만 교체**(I셀 메모에 `(원래정답=X)`) → 채점만 정정, 스포일러 없음.
  - **표시만**(문제 지울것 등): A열 메모 + 행 주황.
  - 멱등(재실행 안전): 이전 표시 청소 + 정답 원복 후 재적용.
  - 검증: 나윤희(동백고2) 완료. 동백고2 전체 확대는 대기.
- **Apps Script 버전**(시트에서 바로 클릭): `완성(코드)\AppsScript_오류표시\` (clasp, '오늘할일' 스프레드시트에 bound). onOpen 메뉴 `🛠 오류표시`. push 완료. **첫 실행 시 브라우저 권한 승인 1회 필요.**

## 2. 학생시트동기화 안전 강화  (데이터손실 재발 방지)
- 어제 동백고2 part1 등 학생 입력/오답 시트 **데이터 손실** 발생(원인: '답지 생성=재생성' 모드가 B열 학생입력 덮어씀). → **시놀로지 예전 버전으로 복구 완료**.
- **학생시트동기화.py** 개선:
  - 덮어쓰기 **전에 직전 백업을 `학생별\_버전\{이름}\{이름}_타임스탬프.xlsx`** 로 보관(학생당 최근 30개). → 손실 후 동기화가 돌아도 좋은 백업이 사라지지 않음.
  - 학생입력(Part1~4 B열) 급감 시 **⚠️ 손실 의심 경고**.
  - **숨김 탭 제외**(속도) — 기본 보이는 탭만. 전체는 `PEDU_SYNC_INCLUDE_HIDDEN=1`.
- 교훈: openpyxl `read_only` 모드의 `.cell()` 랜덤접근은 값 오독 → 카운트는 `iter_rows` 사용.
